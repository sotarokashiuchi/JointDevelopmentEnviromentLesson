# 共同開発環境を構築しよう！
## 概要
- 共同開発環境を構築しようという講座の資料

## 講義の目的
- 自分でプログラミングを学べるようになる
- 学びたいプログラミング言語の環境を構築することができるようになる
- 複数人で共同開発を行う環境を構築できるようになる

## 講義の内容
- テキストエディタの導入方法
- 言語プロセッサの導入方法
- バージョン管理システム(Git)の導入方法
- GitHubの導入方法、使い方

## 講義の対象者
- 1つ以上のプログラミング言語を学んだことがある人
- プログラミング言語に触れたことがある人
